    Enumeration ToolExecutionEvent

    Events emitted by the ToolExecutionManager

    These events provide detailed information about the lifecycle of tool executions, allowing applications to track, display, and respond to tool operations.

    Example

    import { ToolExecutionManager, ToolExecutionEvent } from '@qckfx/agent';

    // Listen for tool execution events
    toolExecutionManager.on(ToolExecutionEvent.COMPLETED, (data) => {
      console.log(`Tool ${data.execution.toolName} completed with result:`, data.execution.result);
    });
